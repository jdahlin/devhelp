GtkHScrollbar

Name

GtkHScrollbar -- A horizontal scrollbar

Synopsis

#include <gtk/gtk.h>

GtkWidget*  gtk_hscrollbar_new              (GtkAdjustment *adjustment);

Object Hierarchy

  GtkObject
   +----GtkWidget
         +----GtkRange
               +----GtkScrollbar
                     +----GtkHScrollbar

Args

  "adjustment"           GtkAdjustment        : Read / Write / Construct

Description

The HScrollbar widget is a widget arranged horizontally creating a scroll bar. See GtkScrollbar for details on scrollbars. GtkAdjustment pointers may be added to handle the adjustment of the scrollbars or it may be left NULL in which case one will be created you. See GtkAdjustment for details.
