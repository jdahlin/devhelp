GtkVScale

Name

GtkVScale -- a vertical slider widget for selecting a value from a range.

Synopsis

#include <gtk/gtk.h>

GtkWidget*  gtk_vscale_new                  (GtkAdjustment *adjustment);

Object Hierarchy

  GtkObject
   +----GtkWidget
         +----GtkRange
               +----GtkScale
                     +----GtkVScale

Args

  "adjustment"           GtkAdjustment        : Read / Write / Construct

Description

The GtkVScale widget is used to allow the user to select a value using a vertical slider. A GtkAdjustment is used to set the initial value, the lower and upper bounds, and the step and page increments.

The position to show the current value, and the number of decimal places shown can be set using the parent GtkScale class's functions.
