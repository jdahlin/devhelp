Setting up on non-automake/autoconf systems

This is quite a bit harder. You will also need to setup xml-i18n-tools which is non-trivial. Setting up things without schema generation and without native language support can be achieved by just using the gnome-config script with the arguments pong, pong_bonobo, pong_glade and pong_manual. I highly recommend using automake and autoconf however for anything being distributed.
