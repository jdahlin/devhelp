Using PonG-Edit

Running Pong-Edit

The best way to start working with PonG is through the pong-edit program. That way you can get a feel for what PonG can do for you by experimenting.

So let's start making a file. Run "pong-edit" from the command line or use the GNOME "Development/Pong-Edit" menuitem. You will get a window with a bunch of notebook panes that represent the different parts of the PonG file.
