The pong-tool

This is a simple program that will take a .pong file on the command line and create a dialog for this file. This way you could setup perhaps a .pong file for some frequently changed settings and run this from a script. This could prove to be a good tool for system administrators as well.

In addition, if you use this together with gconftool, you could use GConf and PonG directly from a shellscript just like in any other application.
