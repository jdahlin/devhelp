GtkFileSelection.get_filename

instance.get_filename();

Returns:

currently-selected filename in locale's encoding

This function returns the selected filename in the C runtime's multibyte string encoding, which may or may not be the same as that used by GTK+ (UTF-8). To convert to UTF-8, call g_filename_to_utf8(). The returned string points to a statically allocated buffer and should be copied if you plan to keep it around.
