Class GtkTextMark

Ancestry

+-- GObject
  +-- GtkTextMark

Methods

GtkTextMark.set_visible

instance.set_visible(setting);

setting

visibility of mark

Sets the visibility of mark; the insertion point is normally visible, i.e. you can see it as a vertical bar. Also, the text widget uses a visible mark to indicate where a drop will occur when dragging-and-dropping text. Most other marks are not visible. Marks are not visible by default.

GtkTextMark.is_visible

instance.is_visible();

Returns:

TRUE if visible

Returns TRUE if the mark is visible (i.e. a cursor is displayed for it)

GtkTextMark.get_name

instance.get_name();

Returns:

mark name

Returns the mark name; returns NULL for anonymous marks.

GtkTextMark.get_deleted

instance.get_deleted();

Returns:

whether the mark is deleted

Returns TRUE if the mark has been removed from its buffer with GtkTextBuffer.delete_mark(). Marks can't be used once deleted.
