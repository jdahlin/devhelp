Class GtkTextTagTable

Ancestry

+-- GObject
  +-- GtkTextTagTable

Constructor

GtkTextTagTable();

Returns:

a new GtkTextTagTable

Creates a new GtkTextTagTable. The table contains no tags by default.

Methods

GtkTextTagTable.add

instance.add(tag);

tag

a GtkTextTag

Add a tag to the table. The tag is assigned the highest priority in the table.

GtkTextTagTable.remove

instance.remove(tag);

tag

a GtkTextTag

Remove a tag from the table. This will remove the table's reference to the tag, so be careful - the tag will end up destroyed if you don't have a reference to it.

GtkTextTagTable.lookup

instance.lookup(name);

name

name of a tag

Returns:

The tag, or NULL if none by that name is in the table.

Look up a named tag.

GtkTextTagTable.foreach

instance.foreach(func, data);

func

a function to call on each tag

data

user data

Calls func on each tag in table, with user data data.

GtkTextTagTable.size

instance.size();

Returns:

number of tags in table

Returns the size of the table (number of tags)
