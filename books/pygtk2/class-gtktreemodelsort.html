Class GtkTreeModelSort

Ancestry

+-- GObject
  +-- GtkTreeModelSort (implements GtkTreeModel)

Methods

GtkTreeModelSort.set_model

instance.set_model(child_model);

child_model

A GtkTreeModel, or NULL.

Sets the model of tree_model_sort to be model. If model is NULL, then the old model is unset.

GtkTreeModelSort.get_model

instance.get_model();

Returns:

the "child model" being sorted

Returns the model the GtkTreeModelSort is sorting.

GtkTreeModelSort.convert_path

instance.convert_path(child_path);

child_path

A GtkTreePath, relative to the child model.

Returns:

A new path, or NULL if child_path does not exist in tree_model_sort ->child_model.

Converts the child_path to a new path, relative to the sorted position. In other words, the value found in the tree_model_sort ->child_model at the child_path, is identical to that found in the tree_model_sort and the return value.
