Methods

GtkTreeSelection.set_mode

instance.set_mode(type);

type

The selection type.

Sets the selection type of the selection. If the previous type was GTK_TREE_SELECTION_MULTI and type is GTK_TREE_SELECTION_SINGLE, then the anchor is kept selected, if it was previously selected.

GtkTreeSelection.set_select_function

instance.set_select_function(func, data);

func

The selection function.

data

The selection function's data.

Sets the selection function. If set, this function is called before any node is selected or unselected, giving some control over which nodes are selected.

GtkTreeSelection.get_user_data

instance.get_user_data();

Returns:

The user data.

Returns the user data for the selection function.

GtkTreeSelection.get_selected

instance.get_selected(model, iter);

model

A pointer set to the GtkTreeModel, or NULL.

iter

The GtkTreeIter, or NULL.

Returns:

TRUE, if there is a selected node.

Sets iter to the currently selected node if selection is set to GTK_TREE_SELECTION_SINGLE. iter may be NULL if you just want to test if selection has any selected nodes. model is filled with the current model as a convenience. This function will not work if you use selection is GTK_TREE_SELECTION_MULTI.
