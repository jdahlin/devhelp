Terminal properties

The terminal has a large number of properties which can be set via object methods.

  • zvt_term_set_size - set absolute terminal size.

  • zvt_term_set_blink - enable/disable blinking cursor.

  • zvt_term_set_bell - enable/disable bell.

  • zvt_term_get_bell - read bell state.

  • zvt_term_set_scroll_on_keystroke - enable/disable auto-scroll on keystroke

  • zvt_term_set_scroll_on_output - enable/disable auto-scroll on output

  • zvt_term_set_del_key_swap - enable/disable DEL-BS key swap.

  • zvt_term_set_wordclass - set characters which match words in selections

  • zvt_term_set_color_scheme - set the color palette

  • zvt_term_set_default_color_scheme - restore default color palette.

  • zvt_term_set_scrollback - set scroll-back buffer size.

  • zvt_term_set_fonts - set fonts.

  • zvt_term_set_font_name - set bold/normal font by name.

  • zvt_term_set_background - set background picture/transparency options.

  • zvt_term_get_capabilities - query compiled capabilities.

These commands change the behavior at runtime.

  • zvt_term_show_pointer - Make the mouse pointer visible.

  • zvt_term_hide_pointer - Make the mouse pointer invisible.
